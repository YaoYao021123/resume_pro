# 原始工作材料说明

## 用途

将你在实习/工作期间产出的原始文件放在这里。生成简历时，Agent 会优先读取这些原始材料（而不是你在 `experiences/` 里自己总结的描述），从中提取更真实、更有说服力的业绩数据和工作细节。

## 目录结构

每家公司建一个子目录：

```
work_materials/
├── 字节跳动/
│   ├── 用户增长分析报告.md
│   ├── 项目复盘.md
│   └── 周报汇总.md
└── 麦肯锡/
    ├── 客户提案.md
    └── 市场调研报告.md
```

## 支持的文件类型

- `.md` / `.txt` — 文字记录、工作日志、分析报告
- 文件内容建议是**纯文本**（Agent 无法读取 Word/Excel/PDF 二进制格式）
- 如果原文件是 Word/Excel，可以复制关键内容粘贴到 `.md` 文件中

## 注意事项

- 如果文件为空或格式不对，Agent 会自动跳过，转而使用 `experiences/` 中的描述
- 原始材料**不需要写得很正式**，工作日记、草稿、碎片记录都可以放进来
