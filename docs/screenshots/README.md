# Screenshots

This directory contains screenshots used in the project README.

## Required Screenshots

| Filename | Description | How to Capture |
|---|---|---|
| `01-web-home.png` | Web UI home page | Open `http://localhost:8765` and screenshot the landing page |
| `02-web-profile.png` | Profile editing page | Navigate to the profile tab and capture the form |
| `03-web-experience.png` | Experience management page | Navigate to the experience tab showing the list |
| `04-web-generate.png` | AI resume generation page | Navigate to the generate tab (with company/role fields visible) |
| `05-web-gallery.png` | Resume gallery page | Navigate to the gallery tab showing generated resumes |
| `06-pdf-output.png` | Example PDF output | Open a generated PDF and take a screenshot |
| `07-claude-code.png` | Claude Code CLI usage | Show a terminal with `/resume` command in action |

## Tips

- Recommended width: 1200px (Retina: 2400px)
- Use light mode for consistency
- Crop to content area (no browser chrome needed for Web UI)
- PNG format, optimized with `pngquant` or similar if large

See `docs/take_screenshots.md` for detailed instructions.
